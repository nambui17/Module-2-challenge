# Nam Bui Portfolio Page

## Description

This website was made to display Nam Bui's portfolio. The website was an exercise in javascript using flexbox to arrange items on a webpage. The website also showcases media queries and responsive web design. The website allows projects to be linked to externally and contains contact information and a links to Resume and projects completed by Nam.

## Usage

Interactive website that provides information on Nam's projects and contact information. In the header, on the top right side of the window, there are navlinks connected to the About Me, Work, Resume, Contact Me sections. Clicking on these links will navigate to the relevant place on the web page. The links on the bottom of the page link to Nam's Github page and a spotify playlist. The About me section contains text information about Nam and a picture. The Work section contains cards linked to projects worked on/created by Nam. To navigate to projects, hover over the cards and left-click to open the link to the project.
